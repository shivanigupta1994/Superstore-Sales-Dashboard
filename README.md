# Superstore Sales Performance Dashboard 📊

## Project Overview
This project is an interactive Power BI dashboard designed to analyze retail sales data from 2014-2017. It provides actionable insights into profitability, regional performance, and shipping efficiency.

## Key Insights Discovered
* **Top Category:** Technology leads in both sales and overall profit.
* **Shipping Bottleneck:** Standard Class shipping shows the longest average delivery time (5 days), highlighting an area for supply chain optimization.
* **Regional Strengths:** The West region is the highest contributor to overall revenue.

## Tools Used
* Power BI (Data Modeling, DAX, Data Visualization)
